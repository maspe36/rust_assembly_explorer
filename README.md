<!-- Plugin description -->
View and explore the assembly generated by your Rust code directly inside RustRover. 
Rust Assembly Explorer integrates with cargo-show-asm to display compiler output for 
functions and symbols, helping you understand optimizations and low-level behavior without 
leaving the IDE.

===
Show all of the assembly output for the current library with rust code interweaved
